#  SmartMoney AI

SmartMoney AI is an AI-powered financial assistant that helps users analyze their financial health and receive personalized financial advice using a large language model.



# Features

-  Financial prediction engine (savings, score, SIP)
-  AI chatbot advisor (LLM-based)
-  Future financial projection
-  Risk profiling
- Goal-based recommendations



# Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Flask (Python)
- AI Model: Groq LLM (Llama 3)
